Toast Message 생성 버튼 

        
twCom객체는 TW Css에서 사용하는 global 객체입니다. jquery.fn을 확장하듯이 모듈의 기능을 twCom객체를 통해 확장 할 수 있습니다.

        twCom객체의 toast 메서드는 첫번쨰인자로 출력할 문자열을 받게되고 두번째인자는 메시지를 유지할 시간을 전달하게됩니다.
 첫번째 인자만 주게되는 경우에는 기본시간 2000ms가 적용됩니다.
